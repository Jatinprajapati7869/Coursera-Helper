# Coursera Downloader

A powerful and user-friendly tool to download Coursera courses, complete with a Graphical User Interface (GUI).

## Features

- **GUI Interface**: Easy-to-use graphical interface for managing downloads.
- **Course Downloading**: Download complete courses including videos and materials.
- **Resume Capability**: Resume interrupted downloads.
- **Organization**: Automatically organizes downloaded content into folders.

## Installation

1.  Clone the repository:
    ```bash
    git clone https://github.com/Jatinprajapati7869/Coursera-Helper.git
    cd Coursera-Helper
    ```

2.  Install the required dependencies:
    ```bash
    pip install -r requirements.txt
    ```

## Usage

To start the application, run the following command in your terminal:

```bash
python run.py
```
